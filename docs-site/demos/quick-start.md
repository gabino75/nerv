# Quick Start

Create a project, get AI-powered recommendations, and run your first Claude session.

## 1. Empty Dashboard

When you first open NERV, you see a clean dashboard with the project selector and an empty kanban board.

![Empty Dashboard](/screenshots/demos/quick-start/01-empty-dashboard.png)

## 2. Create a Project

Click **"+ New Project"** to open the project creation dialog.

![New Project Dialog](/screenshots/demos/quick-start/02-new-project-dialog.png)

## 3. Fill in Details

Enter a project name and a detailed goal. The goal acts as a spec — Claude uses it to generate recommendations.

![Project Form Filled](/screenshots/demos/quick-start/03-project-form-filled.png)

## 4. Empty Kanban Board

After creating the project, you land on an empty kanban board ready for tasks.

![Empty Kanban](/screenshots/demos/quick-start/04-empty-kanban.png)

## 5. Ask "What's Next?"

Click the **"What's Next?"** button to open the recommendation panel. Claude analyzes your project and suggests next steps.

![What's Next Panel](/screenshots/demos/quick-start/05-whats-next-panel.png)

## 6. Review Recommendations

Claude returns recommendation cards — each one is an actionable suggestion like creating a cycle or adding a task.

![Recommendations](/screenshots/demos/quick-start/06-recommendations.png)

## 7. Kanban Board with Task

After approving recommendations, tasks appear on the kanban board in the appropriate columns.

![Kanban with Task](/screenshots/demos/quick-start/07-kanban-with-task.png)

## 8. Add a Task Manually

Click **"+ Add Task"** to create tasks yourself. Choose a type, enter a title and description.

![New Task Dialog](/screenshots/demos/quick-start/08-new-task-dialog.png)

## 9. Multiple Tasks

The board now shows multiple tasks organized across workflow columns.

![Kanban Two Tasks](/screenshots/demos/quick-start/09-kanban-two-tasks.png)

## 10. Claude Working

Start a task and Claude begins working in the terminal — reading files, writing code, running tests.

![Terminal Working](/screenshots/demos/quick-start/10-terminal-working.png)
