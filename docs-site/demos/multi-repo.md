# Multi-Repo

Manage multiple repositories within a single NERV project. Connected repos, CLAUDE.md knowledge bases, and git worktrees are all accessible from the dashboard.

## 2. Connected Repositories

Open the **Repos** panel from the More menu to see all connected repositories and their metadata.

![Repos Panel](/screenshots/demos/multi-repo/02-repos-panel.png)

## 3. Knowledge Base

The **Knowledge Base** panel shows CLAUDE.md files from each repo. Claude reads these to understand every codebase's conventions and architecture.

![Knowledge Panel](/screenshots/demos/multi-repo/03-knowledge-panel.png)

## 4. Git Worktrees

The **Worktrees** panel shows isolated git worktrees per task. Each agent works in its own worktree to avoid conflicts.

![Worktrees Panel](/screenshots/demos/multi-repo/04-worktrees-panel.png)

## 5. Task Detail

Click a task to see its full context, including which repo it belongs to and cross-repo dependencies.

![Task Detail](/screenshots/demos/multi-repo/05-task-detail.png)
