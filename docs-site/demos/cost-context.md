# Cost & Context

Track token usage, session costs, and budget across all projects. The Cost Dashboard provides full visibility into AI spend.

## 1. Dashboard

The main NERV dashboard. When a Claude session is active, the Context Monitor bar shows live token count and model info.

![Dashboard](/screenshots/demos/cost-context/01-dashboard.png)

## 2. Cost Overview

Open the **Cost Dashboard** from the More menu. The overview shows summary cards with total cost, token counts, and session stats.

![Cost Overview](/screenshots/demos/cost-context/02-cost-overview.png)

## 3. Cost by Model

The per-model breakdown compares spend across different Claude models (e.g., Sonnet vs. Opus).

![Cost by Model](/screenshots/demos/cost-context/03-cost-by-model.png)

## 4. Budget Progress

The budget progress bar shows spend vs. your monthly limit, helping you stay within budget.

![Budget Progress](/screenshots/demos/cost-context/04-budget-progress.png)

## 5. Cost by Project

Per-project cost allocation identifies which projects consume the most resources.

![Cost by Project](/screenshots/demos/cost-context/05-cost-by-project.png)

## 6. Recent Tasks

The Recent Tasks view shows cost and token usage for each individual task, sorted by recency.

![Recent Tasks](/screenshots/demos/cost-context/06-recent-tasks.png)
