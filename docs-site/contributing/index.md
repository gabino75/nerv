# Contributing

Thank you for your interest in contributing to NERV! Choose a topic to get started:

- **[Development Setup](./development)** — Prerequisites, environment setup, code style, and common tasks
- **[Testing](./testing)** — Unit tests, E2E tests, benchmark testing, and writing new tests
- **[Releasing](./releasing)** — Release process, versioning, packaging, and CI/CD

## Quick Links

```bash
# Clone and setup
git clone https://github.com/gabino75/nerv.git
cd nerv && npm install && npm run rebuild

# Start developing
npm run dev

# Run checks before committing
npm run build && npm run typecheck && npm run test:unit
```

## Questions?

If you have questions about contributing:

1. Check existing [GitHub Issues](https://github.com/gabino75/nerv/issues)
2. Open a new issue for discussion
